# Skills legacy

Ce dossier contient l'ancienne version plate des commandes du vault.

Les emplacements canoniques sont maintenant :

- `.claude/commands/` pour les slash commands Claude Code
- `.claude/skills/` pour les skills locaux Claude
- `.opencode/skills/` pour les skills locaux OpenCode

Fichiers concernes :

- `prime.md`
- `ingest.md`
- `save.md`
- `query.md`
- `lint.md`

Tu peux les conserver comme reference ou modele de migration, mais les versions actives sont celles sous `.claude/` et `.opencode/`.
